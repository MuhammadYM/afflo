# Focus Sound Files Needed

The following MP3 audio files need to be added to the Resources folder:

## Free Tier Sounds (5 sounds):

1. **rain.mp3** - Rain/rainfall ambient sound (loopable)
2. **ocean.mp3** - Ocean waves ambient sound (loopable)
3. **white_noise.mp3** - White noise sound (loopable)
4. **campfire.mp3** - Campfire crackling sound (loopable)
5. **coffee_shop.mp3** - Coffee shop ambience (loopable)

## Requirements:
- Format: MP3
- Duration: 1-5 minutes (will loop)
- Quality: 128kbps minimum
- Seamless loop (no clicks at loop point)

## Free Resources:
- Freesound.org
- YouTube Audio Library
- Pixabay Sounds
- Zapsplat (free tier)

## How to Add:
1. Download audio files
2. Add to Xcode project: Resources folder
3. Ensure files are added to target
4. Verify file names match FocusSoundType enum
